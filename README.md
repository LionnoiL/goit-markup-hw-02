# WEBSTUDIO

## Description

The WEBSTUDIO project is the first homework assignment on HTML markup as part of
the GoIT Fullstack program. The project demonstrates the creation of a semantic
HTML structure for a digital solutions agency website.

## Project Structure

```
goit-markup-hw-01/
├── index.html       # Main HTML file with markup
├── images/          # Directory with team images
│   ├── team-1.jpg
│   ├── team-2.jpg
│   ├── team-3.jpg
│   └── team-4.jpg
└── README.md        # This file
```

## Website Content

### Header

- WEBSTUDIO company logo
- Navigation menu (Features, Team, Portfolio)
- Contact information (email and phone)

### Main

- **Hero section** — main heading and call-to-action button
- **Features section** — list of key company advantages
- **Team section** — team members presentation with photos and job titles

### Footer

- Company logo
- Business description

## Technologies

- **HTML5** — semantic markup
- Proper document structure
- Use of semantic tags (`<header>`, `<main>`, `<footer>`, `<section>`, `<nav>`,
  `<address>`)

## Project Goal

This project is designed to practice the following skills:

- Creating semantically correct HTML markup
- Organizing content into logical sections
- Using anchor links for navigation
- Proper embedding of media elements
- Basic website structure

## Running the Project

Open the `index.html` file in a web browser to view the website.
